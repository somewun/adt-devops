About QandA

Version 0.0.5

A simple Question and Answer revision app converted to a web interface using Flask. Source code can be found here: https://github.com/somewun/adt-devops
This conversion provides a minimal web UI for the same functionality as my orginal app found here: https://github.com/somewun/QandA

User Guidelines:

  • Press Next Q to step through questions.
  • Press Random Q to jump to a random question.
  • Press View Answer to see the answer to the current question.
