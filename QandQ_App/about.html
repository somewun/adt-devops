About QandA

A simple Question and Answer revision app converted to a web interface using Flask. Source code can be found here: https://github.com/somewun/adt-devops This conversion provides a minimal web UI for the same functionality as my orginal app found here: https://github.com/somewun/QandA

Usage notes:

  • Create a CSV file named QandA.csv in the same folder as this script.
  • CSV format: ID,Question,Answer (you can include a header row).
  • Press Next Q to step through questions, Random Q to jump to a random question, and View Answer to see the answer.
